This folder contains datasets associated with the analyses conducted as part of the manuscript by Eskuche-Keith et al. titled "European Marine Protected Areas are well-placed and climate-resilient, but weakly protected".

The folder 'Inputs' contains all of the relevant data files required to run the analyses:
- Shapefile layers for the MPAs
- Raster layers for the species distribution models
- An .rds file the biotic groupings associated with each species
- Shapefiles for study region

The folder 'Outputs' contains all of the data outputs from the analyses:
- .rds files containing the area overlaps between species distributions and MPAs, for current-day and future conditions
- .csv files with the various summary tables used in the manuscript and supplementary materials
