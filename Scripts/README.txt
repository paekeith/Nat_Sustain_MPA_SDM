This folder contains scripts associated with the analyses conducted as part of the manuscript by Eskuche-Keith et al. titled "European Marine Protected Areas are well-placed and climate-resilient, but weakly protected".

The purpose of each script is described below:
- Distribution_overlap_code: Runs the primary analysis of the overlap between species distributions and MPAs
- Calculating total MPA area coverages: Provides code that calculates the areas and relative proportions of pan-European seas and individual EEZs which overlap with MPAs
- Generating overlap summary tables: Provides code that generates summary tables of the coverage of species distributions by MPAs, across all species and by biotic group
- Plotting outputs: Provides code to generate figures used in the manuscript.
